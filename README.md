# Smart Booking Advisor ✈️

## What We Built
An AI-powered flight price predictor that tells you Book Now or Wait.

## Models
- LightGBM Regression → Predicts flight price (R² = 0.9996, MAE = ₹168.80)
- Random Forest Classifier → Book Now / Wait (Accuracy = 93.42%)

## Dataset
- 300,153 Indian domestic flight records
- 6 airlines, 6 cities, 30 routes

## Key Insights
- Prices spike sharply in the last 15 days before departure
- Book 20+ days early for best Economy prices

## Tech Stack
Python, LightGBM, RandomForest, Pandas, Scikit-learn, Matplotlib
